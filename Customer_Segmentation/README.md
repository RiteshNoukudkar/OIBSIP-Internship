# 🧠 Customer Segmentation Analysis using K-Means Clustering

## 📌 Project Overview

This project applies **Customer Segmentation Analysis** using **K-Means Clustering** to identify distinct groups of customers based on their annual income and spending behavior. The goal is to help businesses understand customer patterns and develop targeted marketing strategies.

---

## 🎯 Objectives

- Analyze customer demographic and spending data.
- Perform Exploratory Data Analysis (EDA).
- Identify customer groups using K-Means Clustering.
- Visualize customer segments.
- Generate business insights for decision-making.

---

## 📊 Dataset Information

**Dataset:** Mall Customers Dataset

### Features:

- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

### Dataset Size

- Total Records: 200
- Total Features: 5

## 🤖 Machine Learning

### K-Means Clustering

K-Means clustering was used to group customers based on:

- Annual Income (k$)
- Spending Score (1-100)

### Steps Performed

1. Feature Selection
2. Elbow Method
3. Optimal Cluster Selection
4. K-Means Model Training
5. Cluster Visualization
6. Business Interpretation

### Cluster 1
Customers with moderate income and moderate spending behavior.

### Cluster 2
High-income customers with high spending scores (Premium Customers).

### Cluster 3
Low-income customers with low spending behavior.

### Cluster 4
High-income customers with low spending behavior and strong marketing potential.

### Cluster 5
Low-income customers with high spending behavior.
